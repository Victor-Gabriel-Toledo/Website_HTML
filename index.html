Estrutura básica do HTML

Voltar

Nesta aula eu aprendi como é formada a estrutura básica do HTML e a diferenciar para que serve cada elemento da estrutura, por exemplo a tag html essa tag é o que faz navegador entender que tudo o que está dentro dessa tag é um html, outra tag que forma a estrutura é o head que é a cebeça da estrutura e dentro dela estão elementos que carregam antes de chegar para o usuário e também temos a tag body que é o corpo da estrutura e tudo que está dentro dessa tag é o que o usuário pode ver e interagir.

Falando sobre tags

Voltar

Aqui eu falarei sobre o que são as tags e suas características, as tags nada mais são do que elementos usados para estruturar e organizar o conteúdo de uma página web, indicando ao navegador como o conteúdo deve ser exibido. Umas das características de uma tag é sua abertura e fechamento iniciando com < > e fechando com < / >, repare que ao fechar uma tag se coloca uma barra na frente.

Atributos das tags

Voltar

Agora falarei sobre os atributos que as tags podem ter, para exemplificar atributos nada mais são do que propriedades que uma tag pode ter, e há atributos tanto globais onde qualquer tag pode ter este tipo de atributo como por exemplo o id e existem atributos que são específicos de algumas tags como por exemplo o type que é específico da tag input.

Textos

Voltar

Aqui nesta aula foi ensinado sobre as tags para formatação de texto como por exemplo h1 e h2 que compõem o título e o subtítulo e que pode ir até o h6, outro exemplo é a tag p que é usada para indicar o começo e o fim de um parágrafo, também temos a tag blockquote que cria uma citação no seu texto entre outras tags que podem ser usadas para formatar um texto.

Lista ordena e não ordenada

Voltar

Já aqui eu aprendi a diferença de uma lista ordenada e uma lista não ordenada e a como criar uma lista, para criar uma lista primeiro você precisa começar com a tag ol ou ul e dentro destas tags precisa ter a tag li que são os itens da lista, oque difere uma lista ordenada para uma não ordenada é justamente a tag que se usa para criar a lista onde a tag ol define uma lista ordenada ou seja nesta lista cada item será numerado por exemplo:

  1. item 1
  2. item 2
  3. item 3
já a lista não ordenada ela é difinida pela tag ul e seus itens não são numerados como por exemplo:
  • item 1
  • item 2
  • item 3
e essas são as diferenças entre uma lista e outra.
